Writing HTML

Atom is the tool we'll use to write our web pages.

Download Atom here.

You can preview your web pages in Atom using the atom-html-preview package. To install the package in Atom:

  • Open the Atom->Preferences menu item.
  • Click the "+Insall+" tab
  • Type "atom-html-preview" into the search box.
  • Click on the "Install" button

    • There is now a menu item Packages->Preview HTML->Enable Preview. Use this to see how your page will look.
